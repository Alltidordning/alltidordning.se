Vanliga frågor

Vem betalar städning och linne?

Det kan debiteras gästen eller dig som värd. Vi rekommenderar pass‑through: gästen betalar och vi kvalitetssäkrar.

Kan jag pausa under vintern?

Ja. Säg till i god tid så justerar vi kalender, prissättning och kommunikation.

Hur funkar nycklar?

Kodlås eller nyckelbox är smidigast. Vi kan installera och administrera – eller sköta fysisk utlämning.

Jour – är ni tillgängliga dygnet runt?

I Fullservice ingår 07–22. Dygnet‑runt kan erbjudas som tillägg vid behov.

Hur snabbt svarar ni gäster?

Målsatt svarstid < 1 timme dagtid. Automatiska mallar och bevakning säkrar snabba svar.

Hur binder jag upp mig?

Start med 3 månaders bindtid, därefter löpande månadsvis uppsägning.
